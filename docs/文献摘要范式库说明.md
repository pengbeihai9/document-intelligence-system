# 文献摘要范式库说明

`academic_abstract_examples.jsonl` 用于存放高质量论文摘要样例。系统检测到上传内容是论文或研究报告时，会从该文件中选择相近样例，作为大模型生成学术摘要时的写作风格参考。

每行是一个 JSON 对象：

```json
{"title":"论文标题","source":"期刊或来源","abstract":"高质量中文摘要文本"}
```

注意事项：

- 样例只用于学习摘要结构、语言密度和表达风格。
- 提示词要求模型不得复制样例中的事实、数据和结论。
- 可以继续追加高质量论文摘要，数量越多，风格参考越稳定。
- 如果需要真正训练或微调模型，需要准备“论文全文或长文本 + 人工高质量摘要”的配对数据，并使用支持微调的平台。
