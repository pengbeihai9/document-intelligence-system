# 顶刊顶会文献采集与下载说明

## 目标

本项目需要构建一批高水平文献，用于摘要测试、摘要评测集建设和后续可能的微调数据准备。采集原则是只使用公开 API 和合法开放获取 PDF，不绕过出版社付费墙。

## 推荐流程

### 0. 一键采集并下载

如果只想直接跑完整流程，可以使用：

```powershell
python tools\collect_and_download_top_papers.py `
  --query "vision transformer" `
  --limit 100 `
  --download-limit 30 `
  --mailto your_email@example.com `
  --skip-existing
```

输出目录：

```text
data/papers/openalex_top_papers.csv
data/papers/openalex_top_papers.jsonl
data/papers/pdfs/
data/papers/download_manifest.csv
```

### 1. 采集顶刊顶会元数据

使用 OpenAlex 按关键词、顶刊顶会列表和引用量筛选论文：

```powershell
python tools\collect_openalex_top_papers.py `
  --query "transformer bankruptcy prediction" `
  --out data\papers\openalex_top_papers.csv `
  --jsonl data\papers\openalex_top_papers.jsonl `
  --limit 100 `
  --mailto your_email@example.com
```

输出包含标题、年份、期刊/会议、DOI、引用量、开放获取状态、PDF 链接和摘要。

### 2. 下载开放获取 PDF

将可合法访问的开放 PDF 统一下载到一个文件夹：

```powershell
python tools\download_open_access_pdfs.py `
  --input data\papers\openalex_top_papers.csv `
  --out-dir data\papers\pdfs `
  --manifest data\papers\download_manifest.csv `
  --limit 50 `
  --skip-existing
```

下载后的 PDF 默认保存在：

```text
data/papers/pdfs/
```

下载记录保存在：

```text
data/papers/download_manifest.csv
```

`data/` 已在 `.gitignore` 中排除，避免误提交 PDF、数据库和运行数据。

## 可扩展插件

后续建议继续补充：

- `collect_semantic_scholar.py`：补引用量、相关论文和 `openAccessPdf`。
- `resolve_unpaywall_pdf.py`：根据 DOI 查找合法开放获取 PDF。
- `collect_openreview.py`：采集 ICLR、部分 NeurIPS/ML 会议文献。
- `collect_acl_anthology.py`：采集 ACL、EMNLP、NAACL 等 NLP 顶会文献。

## 注意事项

1. 只下载 OpenAlex、arXiv、OpenReview、ACL Anthology、Unpaywall 等来源提供的开放访问 PDF。
2. 不下载需要登录、订阅或付费访问的出版社 PDF。
3. 不使用 Sci-Hub 或任何绕过访问控制的方式。
4. 下载前建议设置较小 `--limit` 做测试，确认主题和来源合适后再扩大规模。
5. OpenAlex 查询建议填写 `--mailto`，便于礼貌使用 API。
