# 文档智能处理系统：一键运行说明

## 1. 项目位置

```text

```

主要文件：

```text
app.py                         主程序
requirements.txt               Python 依赖

README.md                      GitHub 项目说明
项目说明与运行部署.md           完整项目说明
```

## 2. 本地运行

打开命令行后执行：

```bat
cd /d D:\DESTOP\老师的题目_V2
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

启动成功后访问：

```text
http://localhost:8501
```

## 3. 使用流程

1. 打开网页后先注册账号。
2. 使用账号登录系统。
3. 上传 PDF、Word、图片或扫描版 PDF。
4. 系统自动提取文本，扫描件会执行 OCR。
5. 页面展示自动分类、文本摘要、词频柱状图、词云和提取全文。
6. 侧边栏“我的文档”只显示当前账号上传过的文档。
7. 可下载提取后的文本。

## 5. 公网部署

1. 将本项目上传到 GitHub。
2. 打开 Streamlit Community Cloud：

```text
https://share.streamlit.io/
```

3. 新建应用，选择该 GitHub 仓库。
4. Main file path 填写：

```text
app.py
```

5. 点击 Deploy，部署成功后复制公网访问链接。
